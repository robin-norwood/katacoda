# Using Terraform modules

First, configure AWS for this scenario:

```
aws configure
```{{execute}}

You will be prompted to enter the AWS Key ID and Secret Key provided for this
lab. Use `us-east-1`, or your preferred AWS region, as the region.

```
$ aws configure
